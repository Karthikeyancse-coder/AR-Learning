# AR-Learn

An immersive, Augmented Reality-based learning platform covering disciplines like Chemistry, Biology, and more. AR-Learn provides interactive holographic experiences to revolutionize science education with real-time feedback and intelligent tutoring.

## Features

- **Hyper-Immersive Learning**: Engage with course material using AR capabilities, exploring everything from human anatomy to molecular structures.
- **Dynamic Dashboard**: Track academic progress, manage active courses, view recent activities, and monitor your study streaks.
- **Dedicated Discipline Modules**: Specialized interfaces for focused subjects like Biology and Chemistry.
- **Comprehensive Security & Privacy**: Fine-grained settings offering 2FA support, connected device management, and detailed privacy controls.
- **AI-Powered Diagnostics**: Built-in backend models for intelligent prediction and recommendations tracking your readiness.

## Project Structure

- `/client` - The frontend application built closely with React, Vite, and TailwindCSS to provide a seamless, modern academic UI.
- `/server` - Backend services, APIs, and AI prediction layers (`predict.py`).
- `/content` - Shared educational resources and assets.

## Getting Started

### Prerequisites
- Node.js (for the frontend client)
- Python 3.x (for backend ML/prediction services)

### Client Setup
```bash
cd client
npm install
npm run dev
```
